Инструкция по созданию проекта в github

  • Создать репозиторий в github
  • Создать папку с сайтом на своем компьютере
  • Выполнить git init в созданной папке
  • Скопировать адрес репозитория (remote)
  • Выполнить git remote add origin (то, что скопировано)
  • Создать файлы в проекте
  • Выполнить git add .
  • Выполнить git commit -m "Text soobshenia v commite"
  • Выполнить git push origin master
